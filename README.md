# 🏠 House Rental Platform - Frontend

A modern and responsive House Rental Platform built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Users can browse rental properties, search and filter listings, view house details, and manage their own property listings through a secure authentication system.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kawserali-64/house-rental-client
```

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
